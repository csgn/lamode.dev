# Usage
## Run the service
```bash
$ MODE=dev make up # will use "docker/modes/dev" environment file
```

## Down the service
```bash
$ MODE=dev make down
```

## Show the summary of the compose config
```bash
$ MODE=dev make config
```

## Stop the service
```bash
$ MODE=dev make stop
```
